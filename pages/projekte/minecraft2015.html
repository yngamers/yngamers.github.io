Minecraftprojekt

Wir wollen in einer ausgiebigen Programmier- und Bausession unseren eigenen Server so gestalten, dass aus dem normalen Minecraft ein cooler RPG- und MMOartiger Spielserver im Multiplayerformat wird. Ganz ohne LogMeHamachi und ohne stressiges Modinstallieren wird man dem Server joinen können. Endlich nun ist der Server 24/7!!!! Wer lust am kreatien Bauen von Lanschaften und Städten (natürlich in Minecraft) hat, kann uns jederzeit unter yngamers@gmx.dekontaktieren. Mehr Infos zum Server und Updates kommen in Kürze!
